# 🏇 Hippodrome - Симулятор скачек

## 📝 Описание проекта

Проект представляет собой симулятор скачек с реализацией логирования и тестирования.
Включает в себя классы для управления лошадьми (`Horse`), ипподромом (`Hippodrome`)
и основной логикой приложения (`Main`).

### 🎯 Основные возможности

- Создание и управление лошадьми с именем, скоростью и дистанцией
- Симуляция движения лошадей с использованием случайного коэффициента
- Определение победителя на основе максимальной дистанции
- Полное логирование всех действий и ошибок
- Комплексное тестирование с использованием JUnit 5 и Mockito

## 📋 Функциональные требования

### Класс Horse
- [x] Проверка имени на null и пустые значения
- [x] Проверка скорости и дистанции на отрицательные значения
- [x] Геттеры для всех полей
- [x] Метод `move()` с расчетом дистанции
- [x] Статический метод `getRandomDouble()`

### Класс Hippodrome
- [x] Проверка списка лошадей на null и пустоту
- [x] Метод `getHorses()` возвращает неизменяемый список
- [x] Метод `move()` вызывает движение всех лошадей
- [x] Метод `getWinner()` определяет победителя

### Логирование
- [x] Создание объектов (DEBUG уровень)
- [x] Ошибки (ERROR уровень)
- [x] Информационные сообщения (INFO уровень)
- [x] Ротация логов по дням
- [x] Автоматическое удаление логов старше 7 дней
